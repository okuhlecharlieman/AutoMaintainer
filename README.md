# AutoMaintainer

“An autonomous multi-agent software engineering system that transforms GitHub issues into production-ready pull requests with human-supervised deployment.”

## Deployment Options

### 🥇 Recommended: Render + UptimeRobot (100% Free, Reliable)
- **Frontend**: Render Static Site (Next.js)
- **Backend**: Render Web Service (FastAPI) + UptimeRobot (keeps it awake)
- **No capacity errors** - always available
- **Setup time**: 10 minutes
- **Guide**: [RENDER_DEPLOYMENT.md](RENDER_DEPLOYMENT.md)

### 🏗️ Alternative: Vercel + Oracle Cloud (100% Free, but capacity issues)
- **Frontend**: Vercel (Next.js)
- **Backend**: Oracle Cloud ARM instance (can have capacity errors)
- **Guide**: [VERCEL_ORACLE_DEPLOYMENT.md](VERCEL_ORACLE_DEPLOYMENT.md)
